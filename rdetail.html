Plant Disease Detection Technique using VGG-19


Early detection of plant diseases is crucial for improving crop yield and quality, especially in tomato farming. This project uses a CNN-based approach to identify and classify leaf diseases through image preprocessing, segmentation, and feature extraction. By addressing the limitations of manual inspection in remote areas, the model enables farmers to detect early-stage infections efficiently and access timely, accurate insights — ultimately supporting better crop management and healthier harvests.

Brain MRI Image Segmentation and Classification using PCA-SVM and CNN



Brain tumors are life-threatening abnormal growths that, if undetected, can severely affect the body. This study leverages deep learning and machine learning techniques for early tumor detection using MRI images. A combination of Convolutional Neural Networks (CNN), Support Vector Machines (SVM), and Principal Component Analysis (PCA) enhances feature extraction and classification accuracy. Image segmentation methods like thresholding and contour detection further improve tumor visibility, supporting faster, more reliable medical diagnoses.

Shot Analysis of Batsmen in Cricket Matches using Transfer Learning Techniques


With the rise of deep learning in computer vision, this study applies Convolutional Neural Networks (CNN) and transfer learning to cricket shot classification. Using the Shot-Net dataset of 4,900 images, the model identifies seven distinct batting shots. By customizing VGG-19’s architecture and fine-tuning layers, the approach achieved 98.61% accuracy. This work highlights the potential of AI-driven sports analytics for precise motion recognition and performance analysis in competitive sports.

Tomato leaf disease detection technique

This research presents a VGG-19-based Convolutional Neural Network (CNN) combined with image segmentation for detecting and classifying tomato leaf diseases. The model achieved 88% accuracy with a final loss value of 0.06, effectively identifying early-stage infections. By automating disease detection, this approach offers a reliable and scalable solution for farmers, enhancing crop health management and supporting smarter, technology-driven agricultural practices
