Enhancing Trading Strategies with LLM and Traditional Models: A Business-Driven Sentiment Analysis Approach for Financial Market Forecasting


This project presents a data-driven Forex trading strategy for the EUR/USD pair using sentiment analysis and deep learning. Sentiment is extracted from financial news using FinBERT, DeBERTa, SVM, and Naive Bayes, and combined with technical indicators like RSI and EMA. These features are fed into hybrid models using LSTM to predict short-term price movements. The approach aims to enhance trading accuracy by integrating NLP, machine learning, and time-series forecasting.
